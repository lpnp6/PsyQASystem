程序化思维导图—高效学习的五大元素
思维导图是一套多项学习技巧的组合工具。与其照葫芦画瓢，不如将它彻底分解为最基础的学习技巧，然后一掌握。这种分解的学习方式在NLP中被称为"程序化"。
思维导图的卓越核心
站在NLP的角度来看思维导图，我们可以将其拆解为以下元素：
| 元素 | 相关技巧 |

| - | - |

| 关键词 | 缩句扩句、留白 |

| 分支 | 鱼骨图、金字塔思维 |

| 颜色 | 色彩心理学 |

| 图片 | 超右脑记忆 |

| 结构 | 层级关系、逻辑化 |
要知道，思维导图的任何一个元素都值得你投入充分的时间去探索和感受。我是在画到500多幅的时候才对思维导图略有感觉，当突破1000幅的时候才对之有比较深刻的感悟。
程序升级—如何让思维导图更"简快"
简快思维导图，源自对传统思维导图的本土化升级：通过对具体绘制过程的更新（更多可操作化的技巧和方法的替换），以更简单、快速、高效地学习为核心，将之发展为本土化的思维导图。
| 技术 | 传统思维导图 | 简快思维导图 |

| - | - | - |

| 关键词 | 尽量简单化 | 更加通俗化 |

| 分支 | 尽量简洁化 | 更加扁平化 |

| 颜色 | 尽量丰富化 | 更加简单化 |

| 图片 | 尽量精美化 | 更加实用化 |

| 结构 | 尽量规范化 | 更加灵活化 |
使用简快思维导图时要遵循以下三个原则：


有效果比有道理更重要。


做方法的主人，而不是被它奴役。


优秀之图必有可取之处，多借鉴。
轻松应用—挑选适合你的绘制方式
对于思维导图的学习，很多人通常都从手绘入门，然而，随着互联网的发展，越来越多的思维导图软件受到思维导图爱好者的青睐。
| 绘制方式 | 优点 | 缺点 |

| - | - | - |

| 手绘 | 个性化强、创意丰富 | 耗时、不易修改 |

| 电脑软件 | 制作快速、易修改 | 个性化弱、创意受限 |

| 手绘+软件 | 结合了手绘的创意和软件的便捷 | 需要一定的电脑基础 |
如果你还迷茫于使用何种方式画图，不妨听自己内心的声音。我目前使用最多的导图方式是第三种—先手绘梳理导图，然后在周末或者闲时使用软件进行优化。
轻松绘制—简快地制作思维导图
以拆解文字段落的方式为例，来呈现简快的思维导图绘制步骤：
步骤一：锁定·中心图
先根据整篇内容的叙述，确定一个中心主题。通常中心图的选择越形象、越具体、越易于联想，就越能达到"便于记忆"的效果。可以直接取书的封面或者书里的图片作为中心图，也可以自己手绘一个简单的图形。
步骤二：展开·关键词
仔细阅读文字段落，找出其中的关键词。关键词应该简单明了、通俗易懂，尽量用一两个字表达一个概念。
步骤三：连接·分支
将关键词以分支的形式连接到中心图上。分支应该简洁明了、扁平化，尽量避免过多的层级。可以适当使用鱼骨图或金字塔思维的方式来组织分支。
步骤四：点缀·颜色
给关键词和分支添加颜色。颜色应该简单大方、易于区分，尽量避免过多的颜色。可以根据关键词的意义选择相应的颜色，或者采用简单的配色方案。
步骤五：强化·图片
给关键词添加简单的图片或符号。图片应该实用明了、易于联想，尽量避免过于精美的图片。可以根据关键词的意义选择相应的图片，或者自己手绘一些简单的符号。
通过这五个步骤，你就可以快速地制作出一张简单明了、易于记忆的思维导图了。记住要做方法的主人，根据自己的需求灵活运用。多练习,相信你一定能够成为思维导图的高手!
