《重塑心灵》：一部心灵旅程的指南
1. 时间线的奇妙之旅 🕰️
在《重塑心灵》这部作品中，李中莹女士以一种独特的方式揭示了我们对时间理解的不同面貌。她告诉我们，有些人的时间线是从左至右，从远至近然后再走远，而有些人的时间线会贯通自己：未来在前面，现在眼下，过去则在身后。然而，这些时间线在未经整理之前，可能是乱七八糟的。

时间线的调整可以使一个人减少焦虑或者提高积极性，这是一种极其强大的工具，或者说是一种无形的直线，它在我们的生活中无处不在。
2. 经验掣：感知世界的窗口 🌈
经验掣是一种强大的工具，它可以帮助我们回忆起过去的情景，重温当时的情绪。当我们面对一首熟悉的乐曲，或者一个熟悉的符号，我们可能会回忆起过去的某个时刻，这就是经验掣的力量。
| -- | -- |

| 
视觉型经验掣
 | 一件物品、一件家具，甚至一个地点或者位置 |

|

 | 校歌、救护车响笛、撞钟声、你的名字 |

|

 | 握手、拍肩膀、吻脸、摸头等身体接触 |

| 
内感觉型经验掣
 | 一些静坐宗派所采用的“心号”，佛教徒心中念的佛号等 |
这些经验掣都是我们感知世界的窗口，它们帮助我们理解世界，理解自己。
3. 拓展视野：美国人对NLP的理解 🌎
在《重塑心灵》这部作品中，李中莹女士引用了美国科罗拉多州政府对神经语言程序学（NLP）的定义，这为我们理解这个理论提供了一个全新的视角。NLP被定义为一套详细可行的模式，关于人类行为与沟通程序的一套模式。虽然它本身并非一套心理疗法，但NLP的重要法则可以被运用于了解人类经验和行为，并使之有所改变。
4. 信念系统：塑造我们生活的框架 💭
信念、价值观和规条统称为“信念系统”。这是一种强大的力量，它可以塑造我们的生活，影响我们的决策。一个人面对世界上种事物的处理态度，所依据的正是他的信念系统。因此，我们的信念系统是我们在这个世界生活下去的内在法则，决定了我们的人生是否成功、快乐。

|

 | 我们对事物的评价，对我们行为的指导 |

|

 | 我们生活中的规则和约束 |
总的来说，《重塑心灵》是一部引人深思的作品，它以独特的方式揭示了我们对世界的理解，对时间的感知，以及我们如何通过信念系统塑造我们的生活。
