《心想事成的艺术》读书笔记 🎨
自我创造的力量 ✨
生活就像一幅精美绝伦的画作,充满了无限的可能。然而,若是总是被动接受,被动地寻找"失败"的原因,我们便很难成为这幅画中的主角。正如本书所言,成为一个"自己争取"的人,才是通往心想事成的关键所在。
当我们主动承担起生活中的一切责任时,我们就拥有了改变命运的力量。不再被动地归咎于外部环境或他人,而是以积极主动的心态去创造属于自己的未来。正如书中所说,"自己争取"的人总是会问自己:"我是如何在生活中创造这个的,创造它的目的又是什么?"这种自我反思的能力,让他们时刻保持清晰的头脑,不断寻找突破的可能性。
| -- -- |

| 生活的态度 | 结果 |

| 被动接受 | 陷入"受他人影响"的怪圈 |

| 主动创造 | 成为"自己争取"的人,心想事成 |
打破旧有模式,开启全新可能 🔥
我们常常会陷入一种"滚轮上的仓鼠"般的循环,不断重复着无法带来改变的行为模式。但正如书中所言,"如果重复旧的行为,你只会得到旧的结果。"因此,我们必须学会问自己:"下次我能做点儿什么不一样的吗?"这个问题,才能帮助我们跳出固有的思维定式,开启全新的可能。
就像书中提到的莎莉的例子,她原本一直使用常规的方式去追求心仪的对象,却收效甚微。直到她尝试了"朝反方向努力",忽视对方,反而引起了对方的注意。这种勇于尝试不同方式的精神,正是"自己争取"的人所具备的。他们不会因为失败而气馁,而是积极地从中吸取教训,寻找下一次突破的机会。
| -- | -- |

| 行为模式 | 结果 |

| 重复旧有行为 | 得到旧的结果 |

| 尝试不同方式 | 开启全新可能 |
学会从"反馈"中成长 🌱
正如书中所说,NLP的理念之一就是"没有失败,只有反馈"。这意味着,当事情进展不顺利时,我们不应将之视为失败,而是应该问自己:"我能从这次经历中学习到什么呢?"
这种积极的学习态度,让"自己争取"的人能够从每一次挫折中汲取养分,不断成长。他们不会被过去的错误所束缚,而是专注于未来可能的改变。正如书中所言,"当大多数人还陷在过去的时候,这种思考方式会让我们放下过去,加速前行。"
| -- | -- |

| 对待挫折的态度 | 结果 |

| 视为失败 | 被过去所束缚 |

| 视为反馈 | 从中吸取教训,加速前行 |
以"自己争取"的心态,开启全新的可能 🌟
综上所述,成为一个"自己争取"的人,不仅需要我们主动承担责任,更需要我们时刻保持积极主动的心态。我们要学会问自己不同角度的问题,从中寻找突破的机会;我们要勇于尝试新的方式,打破固有的思维定式;我们更要学会从每一次挫折中吸取养分,不断成长进步。
只有这样,我们才能真正成为生活的主角,开启一幅充满无限可能的全新画卷。正如书中所言,"有了这种心态,你才会实现自己设定的目标,因为你为自己想要做的事负起了责任,你处事会变得更加灵活。"让我们一起,以"自己争取"的姿态,去创造属于自己的精彩人生!🎨✨
